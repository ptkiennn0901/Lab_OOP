- What happens if users choose “Cancel”? -> It is regarded as choosing "No" and the user will get a message "You've chosen: No".
- How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do” and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help). -> use JOptionPane.showOptionDialog with two specified options(as commented inside the source code).
